## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- 💾 Vercel(for deployment)

## 🧑‍💻 How to Use

### 1. Clone the Repo

```bash
git clone (https://github.com/atulsg88/Atul_Portfolio.git)
cd your-repo-name
````

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the App

```bash
npm start
```

### 4. Build for Production

```bash
npm run build
```
